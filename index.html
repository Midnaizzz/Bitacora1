  • Invención de la WWW
  • Primera comunicación HTTP

Apuntes de la clase

Tim Berners-Lee es un reconocido e importante cientifico informatico de origen britanico al que se le atribuye la invencion de la World Wide Web (WWW).

Estableció la primera comunicación entre un cliente y un servidor usando el protocolo HTTP en diciembre de 1990. En octubre de 1994 fundó el Consorcio de la World Wide Web (W3C) con sede en el MIT, para supervisar y estandarizar el desarrollo de las tecnologías sobre las que se fundamenta la Web y que permiten el funcionamiento de Internet.

Resumen

Berners-Lee no solo innovó una herramienta revolucionaria, sino que también estableció la infraestructura crítica para el funcionamiento de Internet.

Apuntes de la clase

Hipertexto: un texto que no importa el orden en que lo leamos

HTML: lenguaje demarcado de de hipertextos (Hyper Text Markup Language) Define el significado y la estructura del contenido web. Ademas se utilizan otras tecnologias para la apariencia de una pagina se usa css o la funcionalidad con java script.

Este estándar que sirve de referencia del software que interactúa con la elaboración de páginas web en sus diferentes versiones. Define una estructura básica y un código (denominado código HTML) para la presentación de contenido de una página web, que incluye texto, imágenes, videos, juegos, entre otros elementos.

Pasos para hacer un archivo html

  1. Creamos un nuevo archivo
  2. Nos vamos abajo de la pantalla y le picamos en plain text
  3. Escribimos html y lo llamamos "index"
  4. Escribimos html:5
  5. Y por ultimo cambiamos el idioma a español

Resumen

Este estándar es una referencia para el software que crea páginas web, definiendo una estructura básica y un código HTML para presentar contenido como texto, imágenes y videos.

°❀⋆.ೃ࿔*:･ Links ･:*ೃ࿔.⋆❀°

Volver al inicio

Ideas principales

  • Vinculo unidireccional
  • Te lleva a un sitio web o URL especifica
  • El link de imagen hace que una imagen sea como un boton
  • Facilitan la navegacion
  • Mejoran la interactividad y la experiencia del usuario en un sitio web
  • href: Esencial para definir el destino del enlace.
  • target: Controla cómo se abre el enlace (misma pestaña o nueva pestaña).
  • Los colores de los enlaces pueden ser personalizados con CSS para mejorar la estética y la usabilidad.
  • Los estados de los enlaces (visitado, no visitado, activo) ayudan a los usuarios a entender su interacción con el sitio.

Apuntes de la clase

Un link es un vínculo unidireccional dentro de un documento digital, que le permite al lector visitar un sitio web o una dirección URL específica en internet.

Sintaxis de un link a->Apertura, href->Atributo, "url"->Destino, "nombre del link"->Parte visible para el usuario, /a->Cierre

Estados:

  • color azul; es un enlace no visitado
  • color morado; enlace ya visitado
  • color rojo; enlace activo

Alt especifica un texto alternativo que se muestra en lugar de la imagen cuando esta no existe

Links de imagen: "a href="#"/img src="imagen.jpg" alt=""//a"

Links de imagen: "a href="#"target"blank"/"texto de link"//a"

Resumen

Además, los links pueden incluir imágenes o abrirse en una nueva pestaña utilizando atributos adicionales como target="_blank". Los links son fundamentales para la navegación web, mejoran la interactividad y son importantes para la accesibilidad y el SEO.

Volver al inicio

Ideas principales

  • Orden en cascada
  • Los estilos se encadenan
  • Selector: Indica el elemento HTML al que se aplicará el estilo.
  • Propiedad: Define el aspecto que se quiere modificar (por ejemplo, font-family).
  • Valor: Especifica el valor de la propiedad (por ejemplo, arial).

Apuntes de la clase

CSS (Cascading Style Sheets)

Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web

Es para la apariencia de la pagina html y tiene tres formas de poner css en html:

  • Hojas de estilo incrustada: se coloca con una etiqueta "style" es adecuada para un solo documento.
  • Hoja de estilo en linea: puede usarse en linea mediante el atributo "style", se puede aplicar en cualquier elemento html incluido en body, funciona si vas a usar varios documentos
  • Hoja de estilo externa: se enlaza mediante un elemento "link" de html y se coloca en la cabecera "head"

Sintaxis de una regla CSS:

#principal->selector, {font-family:->propiedad, arial;}->valor

El orden de prioridad de las tres formas es: primero hojas de estilo en linea, luego hojas de estilo incrustada, y por ultimo hoja de estilo externa

Resumen

CSS es esencial para crear páginas web atractivas y funcionales, permitiendo a los desarrolladores web diseñar y personalizar la apariencia de sus sitios de manera eficiente.
